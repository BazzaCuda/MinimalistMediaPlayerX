[Help Page: Aspect Ratio](Aspect_Ratio)<br />

| Control | Action |
| ------- | ------ |
| **Left click** the window and hold | the window can be dragged/repositioned |
| **Left double-click** the window   | toggle fullscreen mode |
| **Right single-click** the window  | pause/resume playback of audio/video. pause/resume slideshow of images |
| grab bottom edge and drag          | resize the window - for videos, **MMP** will maintain the aspect ratio of the video |
| grab bottom corner and drag        | resize the window - for videos, **MMP** will maintain the aspect ratio of the video |
